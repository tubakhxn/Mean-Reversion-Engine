# Mean Reversion Trading Engine using Z-Score

This project is a statistical trading engine that detects and trades price deviations from fair value using mean reversion logic and z-score signals. It is designed for educational and research purposes, demonstrating how to build a clean, modular, and visually appealing mean reversion trading system in Python.

## What is this project about?
- Computes rolling mean, standard deviation, and z-score on historical price data
- Generates buy/sell/exit signals based on z-score thresholds
- Backtests the strategy and calculates performance metrics (returns, win rate, drawdown)
- Visualizes price bands, z-score oscillations, and equity curve
